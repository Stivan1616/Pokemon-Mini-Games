# Pokemon Types Game - Beta 

Un juego interactivo para adivinar los tipos de Pokémon, con diseños modernos y retro.

## Características

- **Dos Temas Visuales**:
  - **Moderno**: Diseño limpio, glassmorphism y animaciones suaves.
  - **Retro**: Estilo pixel-art inspirado en la Pokedex clásica.
- **Modo de Juego**:
  - Adivina los tipos del Pokémon mostrado.
  - Sistema inteligente que evita repetir combinaciones de tipos ya vistas.
  - Rachas de victorias con anuncios especiales (cada 5, luego cada 10).
- **Lobby**:
  - Selección de modo de juego (actualmente "Modo Aleatorio").
  - Espacios para futuros modos de juego.
- **Progreso Seguro**:
  - Botón de "Home" con confirmación de seguridad para evitar perder rachas activas.

## Cómo jugar

1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en tu navegador web favorito.
3. ¡Empieza a adivinar!

## Tecnologías

- **HTML5**
- **CSS3** (Variables CSS, Flexbox, Grid, Animaciones)
- **JavaScript (ES6+)**
- **PokeAPI** (para datos de Pokémon en tiempo real)
- **Canvas Confetti** (efectos visuales)
